# Lineamientos UI / UX

## Concepto visual

Educación con enfoque agropecuario, moderno, accesible y centrado en la comunidad.

## Paleta de colores

- Verde principal: #2E7D32
- Verde oscuro: #1B5E20
- Verde claro: #66BB6A
- Naranja acento: #F57C00
- Gris oscuro: #263238
- Gris claro: #F5F7FA
- Blanco: #FFFFFF

## Tipografía

- Títulos: Montserrat
- Texto: Inter

## Componentes

### Botones
- Primario: verde
- Secundario: naranja
- Outline: borde verde

### Cards
- Fondo blanco
- Bordes suaves
- Sombra ligera

### Tablas
- Limpias
- Con hover suave
- Encabezado gris claro

### Formularios
- Inputs con borde gris
- Focus verde
- Labels visibles

## Layout

- Header con menú principal
- Hero destacado en home
- Grid de tarjetas
- Sidebar en páginas internas
- Footer institucional completo

## Responsive

- Diseño mobile-first
- Menú hamburguesa
- Botones grandes
- Cards en columna

## Accesibilidad

- Alto contraste
- Texto legible
- Navegación clara
- Formularios accesibles

## Iconografía

- Estilo lineal
- Uso consistente
- Librerías recomendadas: Heroicons o Lucide
