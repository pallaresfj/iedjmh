# Instrucciones del proyecto

## Stack oficial
- Laravel 12
- Filament 5.x
- Livewire 4
- Tailwind CSS
- MySQL
- PHP 8.3+

## Reglas
- Antes de programar, revisar el estado actual del proyecto.
- No duplicar código ni crear estructuras paralelas innecesarias.
- No eliminar archivos sin justificarlo.
- Antes de cambios grandes, proponer plan.
- Mantener código limpio, modular y mantenible.
- Usar nombres consistentes.
- Priorizar accesibilidad, responsive design y claridad institucional.
- Separar claramente frontend público y panel administrativo.

## Contexto funcional
Este proyecto es el sitio web institucional de la IED Agropecuaria José María Herrera de Pivijay, Magdalena.
